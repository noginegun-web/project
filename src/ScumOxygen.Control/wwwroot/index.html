Инструкция

  1. Скопируйте DLL и папку ScumOxygen в каталог сервера.
  2. Запустите сервер с -fileopenlog.
  3. Откройте локальную панель, укажите IP:PORT и API ключ.

Команда запуска

SCUMServer.exe -log -fileopenlog
Проверьте порт и ключ в runtime.json
